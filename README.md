# MERN BootCamp by LCO

In this created a E-Commerce Website for selling T-Shirts

## Versions

1. **Node:** v18.13.0
2. **Express:** v4.17.1
3. **DataBase:** MongoDB 6.0.4
4. **React:** v18.2.0
   Other Requirements can be found in package.json
